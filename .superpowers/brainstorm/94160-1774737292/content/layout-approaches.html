Overall layout approach

Three ways to structure the dashboard for a cleaner, step-by-step workflow

A

Command Bar (recommended)

A persistent horizontal bar sits above every section showing a Scrape All button + per-section status pills (scraped / generated / failed). Each section page gets a clear 3-phase layout: ① Scrape → ② Review & Pick → ③ Generate. Composer is the final nav item.

B

Status Sidebar

The sidebar expands to show live state per section (not scraped / scraped / output ready) with a Scrape All button at the top of the sidebar. Section pages are cleaned up but no explicit step numbering — just cleaner visual grouping of actions.

C

Build / Publish Modes

A top-level mode toggle: "Build" mode works section by section (with Scrape All available), "Publish" mode opens the Composer full-screen. Each section in Build mode shows a compact card with its status — clicking opens a drawer/modal to work on it.
