# Flat White

Newsletter production pipeline for Flat White by The Aussie Corporate. Pulls signals, classifies editorial content, and assembles the weekly newsletter — all from a browser dashboard.

## Setup (one time)

1. Clone this repo
2. Drop the `.env` file you were sent into this folder
3. Double-click **Setup Flat White** — wait for it to finish (~2-3 min)

## Each week

1. Double-click **Start Flat White** — your browser opens automatically
2. Click **Reingest** to pull fresh data (~2-3 min)
3. Review and approve items in the dashboard
4. Click **Assemble** to build the newsletter HTML
5. Close the terminal window when you're done

## Requirements

- macOS
- Python 3.11+ (the setup script installs it via Homebrew if missing)
- The `.env` file (ask your team — never commit this)
